# 核算通 · 集团经营核算平台 Demo

## 打开方式

在浏览器中打开 `index.html`。如果页面已经打开，请按 `Ctrl + F5` 强制刷新加载最新版本。

## 推荐演示路径

1. 在“高层汇报中心 → 经营驾驶舱”查看高层经营结论、预算完成率和风险提示。
2. 在“高层汇报中心 → 经营分析摘要”查看可直接用于月度汇报的文字摘要。
3. 在“高层汇报中心 → 产品线预警”查看红黄绿状态和建议动作。
4. 在“高层汇报中心 → 预算达成”查看预算 vs 实际、完成率和偏差。
5. 在“经营核算工作台”查看集团总营收、总成本、人工成本、公共分摊费用、利润和利润率。
6. 在“数据采集中心”查看营收账单、成本账单、钉钉费用、匹配规则、异常数据和采集日志。
7. 在“产品线核算”查看每条产品业务线的营收、直接成本、人工成本、部门费用、公共分摊费用和利润。
8. 在“成本管理”查看成本工作台、直接成本、人工成本、费用成本和公共成本池。
9. 在“薪酬核算 → 薪资计算工作台”一键计算全部公司工资，确认薪酬结果进入人工成本。

## 业务口径

- 平台第一版是管理驾驶舱口径，不是正式财务总账系统。
- 高层汇报中心面向决策者，展示结论、预警、预算达成和建议动作。
- 财务分析明细保留数据来源、核算口径和明细追溯。
- 参考每月产品线经营分析表，系统保留毛利、毛利率、费用、净利和净利率口径。
- 产品线利润 = 营收 - 直接成本 - 人工成本 - 部门费用 - 公共分摊费用。
- 业务员工工资直接计入产品线人工成本。
- 财务、行政等共享员工进入公共人工成本池。
- 公共费用和共享人工成本默认按营收占比分摊。
- Demo 数据均为虚构数据，不可用于正式发薪、报税或财务申报。
