# 核算通 · 集团经营核算平台 Demo

## 打开方式

在浏览器中打开 `index.html`。如果页面已经打开，请刷新标签页加载最新版本。

## 推荐演示路径

1. 在“经营核算工作台”查看集团本月总营收、总成本、人工成本、公共分摊费用、利润和利润率。
2. 在“产品线核算”查看每条产品业务线的营收、直接成本、人工成本、部门费用、公共分摊费用和利润。
3. 在“薪酬核算 → 薪资计算工作台”一键计算全部公司工资，确认薪酬结果进入人工成本。
4. 在“费用归集”查看费用如何直接计入产品线或进入公共费用池。
5. 在“分摊规则”查看按营收占比、按人数占比和手工比例三种规则。
6. 在“报表中心”查看产品线利润表、集团经营看板和人工成本分析。

## 业务口径

- 平台第一版是管理驾驶舱口径，不是正式财务总账系统。
- 产品线利润 = 营收 - 直接成本 - 人工成本 - 部门费用 - 公共分摊费用。
- 业务员工工资直接计入产品线人工成本。
- 财务、行政等共享员工进入公共人工成本池。
- 公共费用和共享人工成本默认按营收占比分摊。
- Demo 数据均为虚构数据，不可用于正式发薪、报税或财务申报。
